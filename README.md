# 🌍 Global Phone Intelligence Bot

A powerful Telegram bot built with Python for phone number validation and telecom intelligence analysis.

This bot analyzes international phone numbers and provides useful public telecom metadata including:

- ✅ Country detection
- ✅ Region lookup
- ✅ Carrier/network information
- ✅ Timezone analysis
- ✅ Number validation
- ✅ Number type detection
- ✅ International formatting

---

# 📌 Features

- 📱 Validate phone numbers
- 🌍 Detect country and region
- 📡 Identify telecom carrier
- 🕒 Get timezone information
- 🔎 Detect number type (Mobile, VoIP, Fixed Line, etc.)
- 🤖 Telegram Bot integration
- ⚡ Fast async processing
- 🧠 Clean & beginner-friendly source code

# 📦 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/rootNyx404/OSINT-TELEGRAM-BOT.git
cd phone-intelligence-bot
```

---

## 2️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

# 📄 requirements.txt

Create a file named `requirements.txt`

```txt
python-telegram-bot
phonenumbers
```

---

# 🔑 Setup Telegram Bot

Open the Python file and replace:

```python
TELEGRAM_BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```

with your real Telegram bot token from:

👉 https://t.me/BotFather

---

# ▶️ Run the Bot

```bash
python bot.py
```
